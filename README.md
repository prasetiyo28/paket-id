# Paket . id
  REST API Servis Backend dibangun menggunakan Express.JS dan Redis sebagai data storage

# Requirment
  - Local REDIS on port 6379
  - NPM version 6.9.0

# Instalasi
```sh
$ npm install
```

# Testing
```sh
$ npm run coverage
```
testing menggunakan mocha chai dan nyc untuk coverages

# Menjalankan
```sh
$ npm run start
```
